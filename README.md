# Training Program App

48-тижнева програма тренувань з прогрес-трекінгом.

## Стек

- Vue 3 (Composition API)
- Vue Router 4
- Pinia (state management)
- Vite (bundler)
- localStorage (збереження прогресу)

## Структура програми

| Фаза | Тижні | Опис |
|------|-------|------|
| 1. Фундамент | 1–12 | Full Body 3x, базові рухи, 3×8-12, ~10 сетів/м'яз/тиждень |
| 2. DUP | 13–24 | Хвилеподібна періодизація: сила / гіпертрофія / витривалість |
| 3. Спеціалізація | 25–36 | Full Body + фокус-групи, дропсети, темпо, ~14-16 сетів/м'яз |
| 4. Блокова періодизація | 37–48 | Силовий → гіпертрофійний → піковий блок з тестом 3RM |

Кожен тиждень — 4-денний цикл: Адаптація → Навантаження → Пік → Розвантаження.

Тренування: Пн, Ср, Пт (зал) + Нд (басейн).

## Запуск

```bash
npm install
npm run dev
```

## Білд

```bash
npm run build
```

## Навігація

- `/` — Dashboard (загальний прогрес, поточний тиждень)
- `/weeks` — Всі тижні згруповані по фазах
- `/week/:id` — Тиждень (id = 1-48) з тренуваннями

## Функціонал

- Трекінг виконання вправ (чекбокси)
- Прогрес по тижнях, фазах та загальний
- Автоматичне визначення поточного тижня/дня
- Авторозгортання сьогоднішнього тренування
- Глосарій тренувальних термінів (кнопка ? у хедері)
- Swipe-навігація між тижнями (мобілка)
- Адаптивний layout: мобілка (bottom nav) / десктоп (sidebar)
